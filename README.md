# Micro Apps

A collection of micro applications for sustainability community tools.

## Apps

### Risk Source Library
A curated overview of risk assessment resources verified by community members. Used for due diligence under CSDDD, CS3D, and related legislation.

- `risk-source-library/index.html` — Full standalone page with search, filtering, and interactive cards
- `risk-source-library/circle-embed.html` — Lightweight version optimized for embedding in Circle community posts
